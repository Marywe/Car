Función encargada de rocesar los eventos. Por ejemplo, un evento sería cualquier clase de input o que se cierre la ventana.

Función donde se procesan todos los eventos de input y de ventana. Comprueba el input del coche para llamarlo y hacer que se mueva / frene, controla el input del Reset (pulsar a la R) para volver a empezar con el coche, y por último, cierra el bucle de llamada cuando el programa se cierra.
